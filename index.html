Basic PHP Form Handling

Step 1: Create the Form HTML Structure

<form action="page.php" method="post">
    <label for="fname">First Name:</label>
    <input type="text" id="fname" name="fname"><br>
    <label for="lname">Last Name:</label>
    <input type="text" id="lname" name="lname"><br>
    <input type="submit" value="Submit">
</form>

Step 2: Add Input Fields

Use <label> and <input> tags within the form. The input tag has various attributes like type="text" for text fields and type="submit" for a submit button.

Step 3: PHP Code to Handle the Form Data

<?php
$first_name = $_POST['fname'];
$last_name = $_POST['lname'];
echo "First Name: " . $first_name . "<br>";
echo "Last Name: " . $last_name . "<br>";
?>

Step 4: Adding More Fields (Optional)

<label for="email">Email:</label>
<input type="text" id="email" name="email"><br>

Step 5: Styling the Form

Use CSS to style the form elements.

<style>
    label {
        display: inline-block;
        width: 100px;
    }
    input[type="text"] {
        width: 200px;
    }
</style>

Step 6: Testing the Form

Upload the HTML and PHP files to your server. Access the HTML file in a browser, fill out the form, and submit it. Ensure that the form data is correctly processed and displayed by the PHP script.

Using MAMP to View PHP Files Locally

What is MAMP?

MAMP (Macintosh, Apache, MySQL, PHP) is a free, local server environment that you can install on macOS and Windows. It provides all the tools you need to run WordPress on your desktop PC for testing or development purposes. It is commonly used to set up a local PHP environment.

Steps to Use MAMP for Viewing PHP Files Locally

  1. Download and Install MAMP: Go to the MAMP website, download the installer for your operating system, and install it.
  2. Start MAMP: Open MAMP and click on the "Start Servers" button. This will start the Apache and MySQL servers.
  3. Place Your PHP Files in the MAMP Document Root: By default, the document root for MAMP is set to /Applications/MAMP/htdocs on macOS or C:\MAMP\htdocs on Windows. Place your PHP files in this directory.
  4. Access Your PHP Files: Open a web browser and go to http://localhost:8888/yourfile.php. Replace yourfile.php with the name of your PHP file. You should see your PHP file being executed and displayed in the browser.
